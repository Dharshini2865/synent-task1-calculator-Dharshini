# Simple Calculator - Synent Technologies Internship

A command-line calculator built with Python as part of the
Synent Technologies Python Development Internship (Task 1).

## Features
- Addition, subtraction, multiplication, division
- Coloured terminal output using colorama
- Input validation (handles invalid inputs gracefully)
- Division by zero protection
- Loops until user chooses to exit
- Bonus: Web-based calculator (calculator_web.html) — open in any browser, no installation needed

## How to run
1. Install Python from python.org
2. Install colorama: pip install colorama
3. Run: python calculator.py

## Technologies used
- Python 3
- colorama library
